## Build faissdb:build
```
git clone https://github.com/crumbjp/faissdb.git
cd faissdb/docker
./build.sh
```

### Start container
```
cd docker
./dev.sh start_container
```

### Start process
```
cd docker
./dev.sh start
```

### Rebuild
Rebuild server from current source.

```
cd docker
./dev.sh rebuild
```
